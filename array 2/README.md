filter와 map을 둘 다 사용하는 연습법및 배열메서드를 좀 더 익히기 위해 만들었으나 개인적으로  뭔가 배운 걸 응용하고, 어떻게 하면 좋을까 고민하게 되는게 되게 좋다고 느껴진 파트라고 생각하며, 중요하지만 리드미만 보고 넘길 수 있는 파트여서 다시 보는게 좋을 거 같아, 최대한 간결하게 작성하였다.
